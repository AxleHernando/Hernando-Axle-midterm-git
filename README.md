## Answer the following 
What is git used for?
Git is use to track changes in files and projects. It helps developers manage revisions, collaborate with others, and restore previous versions when needed.

What is the difference between git and github?
Git is the tool that manages version control on a local computer. GitHub is an online platform that hosts Git repositories.

Why is branching useful?
Branching allows developers to change codes without affecting the main project. It makes testing and development safer because changes can be merged only when they are ready.
